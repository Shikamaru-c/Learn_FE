音乐事业

  • 2000年周杰伦依唱片合约内容创作了五十首歌曲后，由吴宗宪钦挑十首，就开始准备制作自己的第一张专辑，吴宗宪提供了新台币四千万元作为专辑筹备的部分资金，于当年11月发行周杰伦首张个人专辑《Jay》，并带周杰伦在自己的节目上大力推销。此专辑有了别于市场的突破曲风。风格融合了R&B、Hip-Hop、古典和中国风，初步建立了周杰伦的“周式”歌手形象。周杰伦在金曲奖颁奖典礼上称:“没有吴宗宪就没有周杰伦”。
  • 2001年9月发行个人第二张专辑《范特西》，专辑名称来自英文“Fantasy”的音译；凭借这张专辑周杰伦在2002年度台湾第13届金曲奖颁奖典礼中斩获“最佳作曲人奖”、“最佳专辑制作人奖”以及“最佳流行音乐演唱专辑奖”等五项大奖。
  • 2002年7月发行个人第三张专辑《八度空间》；9月28日个人首次世界巡回演唱会“THE ONE”于台北市立体育场拉开帷幕。
  • 2003年2月，出道仅三年便接受美国《时代》杂志专访，并成为封面人物；周杰伦是继王菲、张惠妹之后，第三位出现在《时代》杂志封面的华人歌手，并被该杂志赞誉为“新一代亚洲流行天王”；2003年7月，全亚洲超过50家电台定7月16日为周杰伦日，并同步首播其第四张个人新专辑《叶惠美》中的主打歌曲《因父之名》，此后每张专辑必定有一首中国风的歌曲。
  • 2004年7月发行个人第五张专辑《七里香》，并凭借该专辑首次获得世界音乐大奖“大中华区最畅销艺人”；同时，根据国际唱片业协会IFPI的统计，专辑七里香销量位居年度世界第42位；同年以歌曲《龙拳》首次亮相春节联欢晚会。
  • 2005年11月发行个人第六张音乐专辑《11月的肖邦》，自这张专辑周杰伦开始作为音乐总监，并尝试执导一些自己歌曲的mv；6月23日，主演的电影处女作《头文字D》全国上映，并凭借该电影一举获得香港电影金像奖，台湾电影金马奖双料最佳新演员奖。
  • 2006年9月发行个人第七张音乐专辑《依然范特西》，专辑中的中国风歌曲《千里之外》更是找来小哥费玉清一起合唱。
  • 2007年4月，由周杰伦与多年合作伙伴方文山、杨峻荣三人共同创办的杰威尔音乐（JVR）有限公司成立；11月2日，发行个人第八张音乐专辑《我很忙》，11月24日周杰伦在中国上海八万人体育场展开个人第三次世界巡回演唱会，与此同时，周杰伦的“THE ONE”、“无与伦比”两次世界巡回演唱会已先后在美国康涅狄格州、洛杉矶、拉斯维加斯、加拿大、澳大利亚、日本东京、新加坡、马来西亚吉隆坡、泰国曼谷、香港、台湾、中国大陆等多个国家地区举办多场。
  • 2008年发行第九张专辑《魔杰座》，2008年1月10日，周杰伦个人自导自演的电影《不能说的秘密》在韩国上映，2月16日，在日本武道馆连开两场演唱会，成为继邓丽君、王菲之后第三位在武道馆开唱的华人歌手；9月，继艺人王力宏、张惠妹、金城武、F4后，成为美国新闻网CNN《TalkAsia》第五位专访的华人明星；同年《青花瓷》歌词成为中国山东、江苏两地高考考题，之前周杰伦的《蜗牛》《上海1943》《听妈妈的话》等歌曲已相继被收录于中小学教科书中，或成为爱国主义教材，而《发如雪》更是被编入日本《通过唱歌记住中国话》中；11月，凭借专辑《我很忙》获得在摩纳哥举办的世界音乐大奖所颁发的“大中华区最畅销艺人”奖，连续三届（2006年、2007年、2008年）获奖，打破之前歌神张学友两连冠的纪录，加上2004年，周杰伦已先后获得四次世界音乐大奖“大中华区最畅销艺人”奖。
  • 2009年7月，个人第三次世界巡演进入尾声，其中澳洲站雪梨演唱会票房空降美国看板。第二；12月入选美国CNN评选的“亚洲最具影响力的25位人物”，并被形容为“非凡艺人”；12月10日，在美国知名电影网站ScreenCrave评选出的“全球十大最值得期待的新秀演员”中，使他成为首位进军好莱坞的台湾艺人。
  • 2010年，诺基亚下载量调查的“全球歌曲下载量最高歌手”为手机销售量。在（2010年1月1日至2010年8月10日）中，手机下载量排名全球第三，仅次于Lady Gaga、麦克·杰克森；同年，入选美国商业杂志《Fast Company》评选的“全球百大创意人物”，与Lady Gaga、Jay-Z成为“唯三”入榜的歌手；2月，自导自演的电视剧《熊猫人》陆续在台湾及中国大陆上映；12月，国际小行星命名中心批准，正式将小行星257248命名为周杰伦星（Chouchiehlun）。
  • 2011年1月14日，周杰伦参演的美国3D电影《青蜂侠》在全美正式上映并于1月28日在台湾上映；2月，周杰伦与NBA篮球巨星科比拍摄雪碧广告，同时创作广告主题曲《天地一斗》，并邀科比出镜一同合拍MV，该歌曲于2011年2月21日在美国加州洛杉矶首发[27]；4月21日，美国《时代》杂志评出2011年“全球最具影响力人物100强”，票选阶段周杰伦位列第二。
  • 2012年5月，登上福布斯中国名人榜第一名； 9月22日，周杰伦在新加坡F1（世界一级方程式锦标赛）赛道开唱。 12月28日，周杰伦发行个人第十二张全新专辑《十二新作》。
  • 2013年5月17日，周杰伦个人第四次世界巡回演唱会“魔天伦”在中国上海奔驰文化中心启动；7月11日，周杰伦个人第二部自编自导自演的电影《天台爱情》全面上映；周董所属的台北新生命小组教会舞团于7月10日在《天台爱情》首映会上演“天台爱情歌舞秀”，吸引众多媒体与民众注意，同时该片也受到了中国知名导演张艺谋的好评。 2014年12月初，周杰伦、李宗盛与中国百老汇达成签约，将与中国百老汇、台湾百老汇及美国百老汇合作之大型音乐剧，采用1999年时根据ABBA乐队作品创作《妈妈咪呀！》大获成功的“口水歌音乐剧”形式 （英文原文为Compilation Musicals，也被称为Jukebox Musicals，以选取某一位知名歌手或演唱团体的音乐作品，再加以一个非常吸引人的全新创作故事，用音乐剧的方式做呈现），将周杰伦和李宗盛的经典歌曲进行结合，创作一部华丽的音乐歌舞剧。
  • 2014年12月26日，发行个人第十三张音乐专辑《哎呦，不错哦》。
  • 2015年1月29日，新婚后的周杰伦在微博透露他将成为《中国好声音》第四季导师。也开始展开电影《惊天魔盗团2》的拍摄。
  • 2015年5月11日，举行记者会宣布担任电影《一万公里的约定》监制，预计2016年夏季全球上映。 2015年6月2日，《中国好声音4》举行发布会，周杰伦透露由他领导的团队选手中，官方已决定保留一名席位参演正在筹备中的音乐歌舞剧。而周杰伦也将担任该片总导演。
  • 2015年9月22日，周杰伦将加盟《功夫熊猫3》的配音，而他饰演的角色为五位功夫大师之一的金猴。随后，在同一天的19点38分，“周杰伦歌迷网官方微博 ”确认此消息，写道：“周杰伦正式加盟电影《功夫熊猫3》‘金猴’配音。这也是周杰伦第二次参与配音（第一次是十万个冷笑话）据悉，电影于2016年1月29日于美国和中国上映。
  • 2016年6月24日，发行个人第十四张音乐专辑《周杰伦的床边故事》。
